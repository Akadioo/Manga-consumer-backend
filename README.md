## 📝 Descripción

Este repositorio contiene el código fuente del microservicio **Consumer Backend** para el proyecto **MangaNoStore**, desarrollado como trabajo académico universitario. Está construido utilizando el framework [NestJS](https://github.com/nestjs/nest) con TypeScript, diseñado para integrarse con la arquitectura de servicios separados del catálogo digital.
